# HTML/CSS/JS Starter

Build websites with Claude Code. No frameworks, no build tools, no complexity.

## What is this?

A plain HTML template for building websites with Claude Code inside Ship Studio. Just describe what you want, and Claude handles the code.

## Getting Started

No installation needed. Just open the project in Ship Studio and start building.

## Project Structure

```
index.html       # Homepage
about.html       # About page
styles.css       # All styles
script.js        # All JavaScript
CLAUDE.md        # Instructions for Claude Code
```

## Adding Pages

Create a new `.html` file and link to it from the navigation. That's it.

## Deploy

Deploy to Vercel, Netlify, or any static hosting. No build step required - just upload the files.
